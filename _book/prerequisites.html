Chapter 1 Prerequisites

Welcome to R for Loss Data Analytics. This book provides R code that can help you understand and apply loss models concepts. Within the book, there are explanations available to guide you through the specific R functions used in a certain code chunk so that you can understand what the code is doing and why we need to use such a function. In order to use this book most effectively, it is helpful to learn and understand the following programming tools in R. By having this background knowledge, you can manipulate the given code to suit your application.

Useful programming tools:

  1. Install and call packages within RStudio
  2. Assign values to variables
  3. Construct and select particular elements of
    • Vectors
    • Matrices
    • Data frames
  4. Construct plots
    • Histograms
    • Lines
    • Curves
    • Multiple plots in one output
    • Include labels, x and y limits, color and a legend
  5. Construct and call functions
  6. Construct loops
  7. Construct if else statements
  8. Read and manipulate data
    • Read in files
    • Take a subset of the data
    • Get certain elements of the data
